# Berry Anxious

**Berry Anxious** is a narrative-style browser game about social anxiety, awkward conversations, and emotionally intense fruit.

You play as a winged strawberry navigating a strange orchard filled with floating fruit characters who challenge you with difficult social questions. Every response affects your anxiety level, score, and final outcome.

The game mixes:

* retro pixel-art visuals
* animated character interactions
* timed dialogue choices
* emotional/social decision-making
* surreal humor

---

## Gameplay

As you travel through the orchard, you meet five different fruits:

* Percy the Peach
* Manny the Mango
* Beatrice the Blueberry
* Penelope the Pineapple
* Gerald the Grape

Each character has their own personality and conversation style.
Players answer timed questions while managing an anxiety meter that rises under pressure.

Correct or thoughtful answers help calm anxiety, while rushed or poor responses increase it.

The final score determines your ending rank.

## Running the Game

1. Clone the repository
2. Open `index.html` in your browser

Or run it locally with a simple server:

```bash
python3 -m http.server
```

Then visit:

```text
http://localhost:8000
```
